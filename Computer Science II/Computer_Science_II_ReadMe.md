
# Computer Science II — Program Collection

This folder contains four intermediate-level programming projects written in Python and Java created as part of my **Computer Science II** coursework. These projects build on the foundations of Computer Science I and introduce more advanced concepts such as stateful program design, modular functions, user-driven workflows, and object-oriented programming. Together, these programs demonstrate my ability to design and implement multi-step applications that simulate real-world systems.

---

## Overview of Programs

### **1. Bank Account System (Python)**
A state based banking simulation that allows users to:
- View their current account balance  
- Deposit money
- Withdraw money with validation
- Maintain updated account state across operations 

**Concepts demonstrated:** state management, functions, input validation, looping menus, arithmetic operations 

---

### **2. Compound Interest Calculator (Java)**
A financial modeling program that :
- Accepts principal, interest rate, and number of years 
- Uses the annual compound interest rate formula
- Outputs the results with clear formatting 

**Concepts demonstrated:** Java syntax, mathematical modeling, user input, formatted output

---

### **3. Fast Food Restaurant Ordering System (Python)**
A menu-driven ordering application that lets users purchase  :
- wings
- cornbread
- drinks

The program calculates totals, handles multiple selections, and simulates a simple point-of-sale system

**Concepts demonstrated:** branching logic, menu systems, loops, input handling

---

### **4. Scholarship Manager (Python)**
A task tracking system for managing multiple scholarships. Users can:
- View three pre-loaded scholarships  
- Add new scholarships 
- Add tasks to a scholarship
- Mark tasks as completed 
- Display scholarship as details 

**Concepts demonstrated:** classes, objects, methods, lists, composition, user-driven workflows
